# VSCode

Add [TODO HighLight][1] plugin to mark annotation, there are several annotations are recommended as fllowing:

  • TODO: Annotate thing(s) need to do
  • FIXME: Annotate bug(s) need to fix
  • HACK: Annotate hack(s) here need to notice
  • NOTE: Annotate why this way/method
  • XXX: Annotate bug(s) here probably

Add [EditorConfig for VS Code][2] plugin to make code consistent.

Add [Import Cost][3] plugin to calculate the size of importing package.

enable .vue support setting as follow:

"importCost.typescriptExtensions": [

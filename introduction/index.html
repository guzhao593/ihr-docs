# Build Setup

# 获取项目 Git使用规范 http://10.16.85.133:8090/pages/viewpage.action?pageId=4064727
git clone http://git.meicloud.com/coreHR/core-hr-fe.git

# 设置npm连接sahara前端标准组件库
npm config set @ipeople:registry http://10.18.37.72:7001/

# install dependencies
npm install

# 首次运行执行一次(应该去除)
npm run dll

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# VSCode

Add [TODO HighLight][1] plugin to mark annotation, there are several annotations are recommended as fllowing:

  • TODO: Annotate thing(s) need to do
  • FIXME: Annotate bug(s) need to fix
  • HACK: Annotate hack(s) here need to notice
  • NOTE: Annotate why this way/method
  • XXX: Annotate bug(s) here probably

Add [EditorConfig for VS Code][2] plugin to make code consistent.

Add [Import Cost][3] plugin to calculate the size of importing package.

enable .vue support setting as follow:

"importCost.typescriptExtensions": [
    "\\.tsx?$",
    "\\.vue$"
]

# Details

不用再单独引入变量文件

所有元素都是{ box-sizing: border-box; }

常用变量都在 element-variables.scss & _variables.scss 里，如需要取用 $brands 和 $neutrals 里变量，使用 map-get() & nth()，跟标注稿颜色一一对应

👇 👇 👇

// 我是例子
.primary {
  color: map-get($brands, primary); // 拿到 $--color-primary 变量
}

.border-light {
  color: nth($neutrals, 4); // 拿到 $--border-color-light 变量
}

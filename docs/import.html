    # 引入外部模块

    # 完整引入

    一般我们引入外部模块，如 element-ui ，使用 Vue.use：

    import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

// 如果要进行全局配置, 可以传入一个全局对象: { size: 'small', zIndex: 3000 }
// 具体要看引入模块官方文档是否提供了第二参数进行配置
// Vue.use(Element, { size: 'small', zIndex: 3000 });

new Vue({
  el: '#app',
  render: h => h(App)
});

    # 按需引入

    首先安装 babel-plugin-component：

    npm install babel-plugin-component -D
    1

    然后，将 .babelrc 修改为：

    {
  "presets": [["es2015", { "modules": false }]],
  "plugins": [
    [
      "component",
      {
        "libraryName": "element-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
}

    接下来，如果你只希望引入部分组件，比如 Button 和 Select，那么需要在 main.js 中写入以下内容：

    import Vue from 'vue';
import { Button, Select } from 'element-ui';
import App from './App.vue';

Vue.component(Button.name, Button);
Vue.component(Select.name, Select);
/* 或写为
 * Vue.use(Button)
 * Vue.use(Select)
 */

new Vue({
  el: '#app',
  render: h => h(App)
}

    # 挂载到原型链上

    还有一种方法是将 外部组件库 代理到 Vue 的原型对象上去，例如引入 momentjs：

    import Vue from 'vue';
import moment from 'moment';
import App from './App.vue';

Vue.prototype.$moment = moment;
// 或者 Object.defineProperty(Vue.prototype, '$moment', { value: moment });

new Vue({
  el: '#app',
  render: h => h(App)
}

    对应 Element-ui 也有使用到这种方法，Element-ui 默认将一些在全局经常需要调用到的方法已经挂载到 Vue 的原型链上了：

    import ElementUI from 'element-ui';

Vue.prototype.$loading = ElementUI.Loading.service;
Vue.prototype.$msgbox = ElementUI.MessageBox;
Vue.prototype.$alert = ElementUI.MessageBox.alert;
Vue.prototype.$confirm = ElementUI.MessageBox.confirm;
Vue.prototype.$prompt = ElementUI.MessageBox.prompt;
Vue.prototype.$notify = ElementUI.Notification;
Vue.prototype.$message = ElementUI.Message;

# Tree 树

用于配合后端数据辅助渲染，拓展了远程搜索

# 基本使用

在 el-tree 的基础上，支持了对平行结构数据的解析和渲染，因此不需要自行处理后端数据，直接传入即可。并指定 data-struct 为 parallel，告知组件传入的数据，是一个待处理的平行结构数据，默认支持的是嵌套结构 nest，与 el-tree 一致。对 props 拓展了 id 和 parentId 属性，用于告知节点自身和直接父级的唯一标识符。

# 搜索

如果需要搜索功能，可以把控件放置在 i-tree 中指定 slot="search"，并且让 show-search 为 true，默认为 false。

美的
美的集团
深圳美云智数科技有限公司
慧享云事业部
产品与项目管理部
<template>
  <div class="demo">
    <i-tree
      :data="data"
      :filter-node-method="filterNode"
      :props="{
        id: 'unitId',
        label: 'unitName',
        parentId: 'parentId',
      }"
      data-struct="parallel"
      show-search
      default-expand-all
      ref="tree">
      <el-input v-model="value" placeholder="组织树搜索" slot="search">
      </el-input>
    </i-tree>
  </div>
</template>

<script>
import data from './mock'

export default {
  watch: {
    value (val) {
      this.$refs.tree.elTree.filter(val)
    }
  },

  data () {
    return {
      value: '',
      data
    }
  },

  methods: {
    filterNode (value, data) {
      if (!value) return true
      return data.unitName.indexOf(value) !== -1
    }
  }
}
</script>

# 懒加载

指定 lazy 和 load 即可。（文档暂不支持跨域，因此只展示代码）

暂无数据
<!-- 懒加载 -->
<template>
  <div class="demo">
    <i-tree
      :load="loadNode"
      :props="{
        id: 'unitId',
        label: 'unitName',
        parentId: 'parentId'
      }"
      lazy
      data-struct="parallel"
      ref="tree"
    ></i-tree>
  </div>
</template>

<script>
import Axios from 'axios'

export default {
  data () {
    return {
      loading: false
    }
  },

  methods: {
    // 加载
    fetchOrgTree (id) {
      return Axios.get(`/backend/org/view/${id}/children`).then(res => res.data)
    },

    loadNode (node) {
      this.loading = true
      return this.fetchOrgTree(
        node.data ? node.data.unitId : 0
      ).then(data => {
        this.loading = false
        return data
      })
    }
  }
}
</script>

# 带搜索的懒加载

传入 query-search 搜索方法，返回 [{ label, value }] 结构的数据，其中 value 为节点路径。指定 auto-scroll 为 true，可以在选中搜索选项后，滚动节点所在的位置。

    暂无数据
    <template>
  <div class="demo">
    <i-tree
      v-model="value"
      :query-search="querySearch"
      :load="loadNode"
      :props="{
        id: 'unitId',
        label: 'unitName',
        parentId: 'parentId'
      }"
      lazy
      data-struct="parallel"
      show-search
      auto-scroll
      ref="tree"
    ></i-tree>
  </div>
</template>

<script>
import Axios from 'axios'

export default {
  data () {
    return {
      value: '',
      loading: false
    }
  },

  methods: {
    // 加载
    fetchOrgTree (id) {
      return Axios.get(`/api/org/view/${id}/children`).then(res => res.data)
    },

    // 搜索
    searchOrgNode (name) {
      return Axios.get(`/api/org/view/query?unitName=${name}`).then(res => res.data)
    },

    querySearch (query, data) {
      return this.searchOrgNode(query)
        .then(data => {
          return data.map(el => {
            return {
              label: el.unitName,
              value: el.path
            }
          })
        })
    },

    loadNode (node) {
      this.loading = true
      return this.fetchOrgTree(node.data ? node.data.unitId : 0).then(data => {
        this.loading = false
        return data
      })
    }
  }
}
</script>

    # API

    # 属性

    参数 说明 类型 默认值
    data 展示数据 Array -
    dataStruct 展示数据的结构 String ('nest'/'parallel') 'nest'
    querySearch 远程搜索方法，第一个参数是输入的搜索字段 Function(query) -
    placeholder 远程搜索的输入框占位符 String -
    showSearch 是否隐藏状态 Boolean false
    autoScroll 是否自动滚动至远程搜索选中的节点 Boolean false

    # 事件

    事件名称 说明 回调参数
    path-change 在懒加载场景下，选中节点发生变化的时候触发 共一个参数，即选中节点的路径

    # 方法

    方法名 说明 参数
    setExpandedKeys 指定展开节点 接收一个数组，该参数是节点对应的路径

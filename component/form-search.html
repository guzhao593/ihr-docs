# FormSearch 高级搜索

三列栅格式的表单排列方式，常用于数据表格的高级搜索。

# 基本使用

轻敲回车触发查询，在 @search 事件中执行请求操作。

# 流式标签

通过 label-flex，表单项控件区域会根据其标签宽度自动撑满余下空间。

# 搜索项列数

高级搜索默认 col.span = 8 呈现三列布局。

# 搜索项行数

通过 rowCount 指定展开前的行数。

# 默认展开

# 工具区位置

展开前占用一个表单项的位置，展开后移至最下方独占一行。

# API

# FormSearch 属性

参数 说明 类型 默认值
labelFlex 标签和控件宽度是否流动 boolean false
defaultCollapse 是否默认收起 boolean false
searchOnEnter 是否回车触发查询 boolean true
rowCount 搜索条件默认展开行数 number 1
toolbarPosition 工具区位置 string: right/bottom right

# FormSearch 事件

事件名 说明 回调参数
search 请求 表单数据
reset 重置 -
collapse-change 展开/收齐 是否收起

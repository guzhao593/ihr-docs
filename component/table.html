# Table 表格

用于渲染多级表头、行操作、分页、可编辑单元格

提供了 TableColumn 函数式组件用于辅助渲染，优势在于通过 columns 数据驱动的方式，统一应对一级表格、多级表格、多类型列表、可编辑单元格、行操作栏的渲染。

# 基本使用

columns 和 data 分别表示表头信息和数据信息。

WARNING

prop 是唯一标识符，用于数据匹配，若处于叶子节点的非特殊类型表头则必填。

例如：

  • { children: [], label: '父级' } 非必填
  • { type: 'selection', label: '复选框' } 非必填

# 特殊类型

通过 type 指定特殊列：selection/index/expand。当 type 为 expand 时，可在 render 写入渲染函数实现自定义内容渲染，其中 render 函数的第一个参数是 Vue 提供的渲染方法，第二个参数是由 el-table-column 提供的 scopedSlot 参数。

# 多级表头

在应对多级表格需求时，开发者只需要单层循环，无需自行编写递归或嵌套组件。数据信息仍然为扁平结构，在表头中通过 children 决定是否有下级表头。组件内部通过判断 children 决定是否递归，以此生成一级或多级表格。由于是无状态的函数式组件，因此最终生成的 VNode 中，不会存在 i-table-column 这一实例对象，这有利于 el-table 渲染正确的表头结构。

# 可编辑单元格

通过 editor 配置可编辑单元格信息，经过重新设计的单元格组件对单元格渲染、表单校验、单元格之间的联动有了更好的支持。editing 决定整个表格是否处于编辑状态，默认为 false。在 editor 中可以通过 props.readonly 控制单元格是否处于编辑状态。

通过 type 指定单元格控件的类型：

  1. text 文本（默认）
  2. input 单行表单
  3. inputNumber 计数器
  4. datePicker 日期选择器
  5. select 下拉选择器（指向 Selector 组件)
  6. lookup 通用选择器（指向 Lookup 组件）
  7. switch 开关
  8. button 按钮

# 校验

在 rules 中可直接传入 el-form 所支持的表单校验和自定义校验。

i-table 提供 validate 方法，用于触发所有可编辑单元格的校验。若所有校验通过，则返回 true，否则为 false。

# 动态配置

只需要将配置信息改为函数即可，第一个参数是单元格的实例对象。在函数内部做判断，返回不同的规则，可以很轻松且直观的实现单元格之间联动。

# 自定义

当 type 为 custom 时，通过 render 编写自定义内容。

TIP

在非模版场景中，不支持 Vue 提供的 v-model 指令，需要自行编写 value={} 和 on-input={ vm.emit('input') } 用于实现双向数据绑定。具体配置可参考 Vue 文档，或直接在 JSX 语法中使用 {...vm.vmodel} 实现同样的效果。

常量用 "" 包裹，变量用 {}。

WARNING

custom 模式下，不支持直接在 render 函数同级的属性上，编写 props 以及 on，只能直接写在 JSX 或者 h 函数中。

# 行操作

通过 operator 配置调出，接收一个 Array。使用 tooltip 启用文本提示。其余配置与 toolbar 一致。

同时支持动态配置。

# 分页

show-pagination 接收 Boolean，在 true 时展示分页栏，默认为 false。info 表示分页信息，其中 { pageIndex, pageSize } 分别对应当前页和每页的数量。total 表示数据总量。

与上一代最大的不同在于，i-table 内部不再维护分页信息以及搜索信息，全权由外界控制，组件不参与具体业务。通过在 i-table 上绑定 @page-change 监听分页信息的变化。或者根据需要，绑定 @index-change、@size-change 分别监听 pageIndex 的变化和 pageSize 的变化。

TIP

改变 pageSize 时，pageIndex 默认置为 1，若有特殊的业务需求，可以绑定 @size-change 监听，在回调中更改 pageIndex。

# Table 事件

事件名 说明 回调参数

index-change 当分页索引 pageIndex 改变时触发 当前页
page-change 当分页信息（pageSize/pageIndex）改变时触发 当前页

# Toolbar 工具栏

用于渲染按钮、按钮集合、下拉菜单、弹出层等控件的工具栏

# 基本使用

在 tools 中配置工具集。type 指定渲染类型，默认为 button。

# 按钮集合

type 为 button-group 时渲染按钮集合，在 props 的 items 属性中配置子按钮。

# 下拉菜单

type 为 dropdown 时渲染下拉菜单，同理在 items 属性中配置下拉项。在 dropdown 中配置 el-dropdown 的属性和事件。

# 弹出框

type 为 popover 时渲染弹出框，在 render 中传入弹出框的渲染方法，第三个参数包含了控制弹出框显隐的方法 setVisible。在 popover 中配置 el-popover 的属性和事件。

TIP

在不指定 el-popover 的 value 和 onInput 的时候，需要使用内置的 setVisible 控制显隐。

# 自定义内容

type 为 custom 时支持在工具栏中渲染自定义控件，在 render 中指定渲染函数。

# 收纳

store 为 true，工具将被收纳，并且根据 align 决定收纳后的位置，默认为 left。注意：type: custom 的自定义控件以及 type: 'popover' 的嵌套控件无法收纳。

# 动态配置

在 tools 数组中可以传入函数实现动态配置，函数第一个参数是 Toolbar 的 context 环境，默认指向 Toolbar 的父级。

# API

# Toolbar 属性

参数 说明 类型 默认值
tools 工具栏 array -
align 对齐方式 string: left/right left
filterTool 过滤工具，返回 false 则过滤 function(tool): boolean -
context 上下文环境 object this.$parent
storedOnLimit 设置超出一定数量时自动收纳的阈值 number 0
storedToolProps 左侧收纳区的工具属性 object -
storedRightToolProps 左侧收纳区的工具属性 object -

# Toolbar 事件

事件名 说明 回调参数
visible-tools-update 可视区工具属性变动时触发 visibleTools

# ToolbarItem 属性

参数 说明 类型 默认值
text 文本 string -
itemType 工具类型 string: button/button-group/dropdown/popover/custom button
hidden 是否隐藏状态 boolean false
stored 是否收纳状态 boolean false
dropdown 下拉菜单配置入口 object -
items 子集，适用于 type: button-group/dropdown array -
align 对齐方式 string: left/right left
render 当 item-type 为 custom 时，可传入自定义内容的渲染函数 Function(h, context) -

# FormX 表单

管理表单项的状态和行为，用以录入、校验、提交数据

# 基本使用

基本的表单数据域控制展示。

# 布局

内置 row 和 col 用于表单布局配置，也可通过 wrapper-col 对表单项局部配置。配置参数可参考 element layout。

# 只读

提供 readonly 属性控制只读状态。若需要进入只读状态，请确保 i-form-item 拥有唯一标识 prop。在进入只读状态后，表单项会重置校验状态，无视校验规则，以保障始终可以通过校验。

# 自定义只读视图

并非所有的只读状态都只展示纯文本，因此 i-form-item 提供了 renderDisplayValue 属性和 readonly 作用域插槽用于自定义只读视图。当表单项的值为空时 emptyText 控制显示的文本，默认为 '-'。

# 附加指引

在 append 插槽中追加用户操作指南。

# 描述

使用 renderDescription 描述所填结果的具体信息。

# 边框

带边框和背景颜色列表。当 labelPosition: 'left' | 'right' 时 labelWidth 不能为 'auto'。

垂直带边框和背景颜色的列表。

# 表单权限

通过 grants 字段控制表单项的隐藏和禁用状态。

# 表单联动

通过 @change 捕获表单项的用户更新行为，@model-update 捕获表单项的数据更新行为。

# API

# FormX 属性

参数 说明 类型 默认值
title 标题 string -
toolbar 工具栏 array -
labelFlex 标签和控件宽度是否流动，仅在 readonly 和 FormSearch 中有效 boolean false
row el-row 布局 object { type: 'flex' }
col el-col 布局 object -
readonly 控制处于只读状态 boolean false
sourceNode 用于动态生成表单 object -
fieldControlTags 注册表单控件，动态渲染时会给这些控件进行双向数据绑定 array -
displayValueFormatter 注册只读状态下的文本显示 object -
hideRequiredAsteriskOnReadonly 控制在只读状态下是否显示必填星标 boolean false
renderWrapperContent 用于渲染表单项外层容器 function -
code 动态渲染的表单编码 string -
grants 权限控制 object -
fetchSourceNode 获取动态表单数据的方法 function(formCode, formProps) -

# FormX 事件

事件名 说明 回调参数
change 表单项的绑定值由用户更改后触发 被更改的表单项 prop 值，更新后的值
model-update 表单项的绑定值变化时触发 被更改的表单项 prop 值，更新后的值，model 集合
props-reflux 表单动态渲染的配置解析后触发 解析后的表单属性
rendered 表单动态渲染完成后触发 解析后的表单属性

# FormX 方法

方法名 说明 参数
getModel 获取表单的模型 -
setModel 设置表单的模型，详细见下表 model: object, options: object
getFieldValue 获取表单项的值 prop: string
setFieldValue 设置表单项的值 prop: string, value: any
getField 获取表单项的实例对象 prop: string
getFields 获取表单项的实例对象集合 -
getFieldReadonly 获取表单项的只读状态 prop: string
setFieldReadonly 设置表单项的只读状态 prop: string, value: boolean
getFieldHidden 获取表单项的隐藏状态 prop: string
setFieldHidden 控制表单项是否隐藏，第二个参数默认为 true prop: string, hidden: boolean
getFieldDisabled 获取表单项的禁用状态 prop: string
setFieldDisabled 控制表单项是否禁用，第二个参数默认为 true prop: string, disabled: boolean
getFieldGrant 获取表单项的权限状态 prop: string
setFieldGrant 控制表单项权限，第二个参数是权限的配置 { hidden, disabled } prop: string, grant: object

# setModel 选项

参数 说明 类型 默认值
exact 是否精准填充 model 中存在的字段 boolean true
required 是否仅非空值才能填充入 model boolean false

# FormItem 属性

参数 说明 类型 默认值
wrapperCol 外层的 el-col 布局 object -
readonly 控制处于只读状态 boolean false
hidden 控制处于隐藏状态 boolean false
disabled 控制处于不可编辑状态 boolean false
labelHidden 隐藏标题 boolean false
renderDisplayValue 自定义只读状态下的视图 function(h, displayValue) -
renderDescription 自定义描述区域的视图 function(h) -
emptyText 在值为空时的只读状态文本内容 string '-'
elFormItemStyle el-form-item 行内样式 string/object -

# FormItem 插槽

name 说明
append 附加信息

# FormItem 作用域插槽

name 说明
readonly 只读视图，参数为 { displayValue }
append 附加信息，参数为 { displayValue }

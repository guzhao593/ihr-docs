# Dialog 对话框

展示信息的对话框

# 基本使用

通过 v-model 或 visible.sync 的形式控制显示或隐藏对话框。

# 不同尺寸

提供了五种尺寸，可以在不同场景下选择合适的尺寸，默认为 medium。

默认尺寸对应关系：huge 1200px / large 900px / medium 700px / small 500px / mini 350px。若需调整尺寸对应的具体数值，可以在全局配置 $SAHARA.dialog.sizes 中改写。

# 默认工具栏

通过 default-toolbar 指定是否显示默认工具栏，默认为 true。若启用，则在点击「取消」时触发 cancel 事件，在点击「确定」时触发 ok 事件。

注意：当 toolbar 有值时默认工具栏会失效，优先渲染 toolbar 中的工具项。

# 保持活跃

通过 keep-alive 控制被关闭的对话框内容实例是否保留，默认为 false，即关闭即销毁。

# 固定高度

通过 content-height 设置内容区域的高度，若实际内容超出阈值，则会出现纵向滚动条。

# 异步关闭

点击确定后触发 before-ok 钩子，通过 done() 异步关闭对话框，例如提交表单；或执行 done(false) 用于阻止对话框关闭，例如表单校验异常时。

# API

# 属性

参数 说明 类型 默认值
visible 控制是否显示对话框，支持双向数据绑定以及 .sync 修饰符 boolean false
size 对话框尺寸 'huge'/'large'/'medium'/'small'/'mini' 'medium'
toolbar 底部工具栏 array -
defaultToolbar 是否启用底部默认工具栏 boolean true
contentHeight 内容区域高度 string -
keepAlive 是否在组件隐藏时保留在内存中，防止重复渲染DOM boolean false
beforeOk 在 default-toolbar 时点击「确定」触发，可通过 done(false) 阻断确认行为 function(done) -

# 事件

事件名 说明 回调参数
visible-change visible 变更的时候触发，用于双向数据绑定 visible
ok 在 default-toolbar 时点击确定触发 -
cancel 在 default-toolbar 时点击取消触发 -

# 方法

方法名 说明 参数
close 关闭对话框 -
ok 执行「确认」行为 -
cancel 执行「取消」行为 -

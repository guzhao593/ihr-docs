# Selector 选择器

支持外部传入和组件自行获取下拉选项数据，包括字典数据、远程搜索、远程搜索

# 基本使用

适用广泛的基础单选，options 用于放置选项信息，v-model 的值为当前被选中的 options[].value 属性值。

# 基础多选

多选时用 Tag 展示已选项，且默认隐藏第二个及以上的 Tag。

# 数据映射

若 options 上的数值不符合 label/value 字段的要求，可通过 props 将其映射。

# 选项禁用

设定 props.disabled 值，即可禁用该选项。

# 字典数据

通过设置 data-key="dict" 将选项的数据来源指向字典接口，并以 param 作为请求参数，Selector 会调用全局配置的 dictFormatter 执行请求行为。

# 全局配置

远程 API 配置相对统一，因此我们可以将它们统一管理。可以在 $SAHARA.selector.remotes 中全局配置请求信息，就能在业务页面中通过 remote-key 获得相应的能力。

例如此处 remotes 下的配置信息名为 getABC，那么只需要在调用时指定 remote-key="getABC" 就能合并此处的配置信息至 Selector 组件：




 
 
 
 
 
 






Vue.use(Sahara, {
  selector: {
    remotes: {
      getABC: {
        valueKey: 'value',              // 对象数据唯一标识，仅在 value-type="object" 时有效
        props: {},                      // 数据映射
        fetchOptions (param) {},        // 全量请求，仅在 data-key="remote" 时有效
        remoteSearch (query, param) {}, // 按需请求，输入时执行，仅在 data-key="remote" 且 lazy 时有效
        remoteEcho (values, param) {},  // 回显请求，初始时执行，仅在 data-key="remote" 且 lazy 时有效
      },
      // add more remote config...
    }
  }
})

# 远程数据

在初始化以及参数 param 变化时，都会调用 fetchOptions 请求获取新的下拉选项数据。

# 远程搜索

lazy 的状态下，会随着用户输入行为通过 remoteSearch 按需从服务端搜索数据。

# 远程回显

lazy 的状态下，会根据初始状态的默认值发起回显请求 remoteEcho。

# 数据类型

如果绑定值为对象类型，请务必指定 value-key 作为它的唯一性标识，默认为 'value'。

# API

# 属性

参数 说明 类型 默认值
value 绑定值 boolean/string/number/object -
options 下拉选项数据 function -
dataKey 数据来源 'options'/'dict'/'remote' 'options'
remoteKey remotes 对应的配置名称 string -
valueType 绑定值的类型 'object'/'primitive' 'primitive'
valueKey value 唯一标识的键名，绑定值为对象类型时必填 string 'value'
props 选项映射，详细见下表 object -
multiple 是否多选 boolean -
disabled 是否禁用 boolean -
clearable 是否可以清空选项 boolean true
collapseTags 多选时是否将选中值按文字的形式展示 boolean true
lazy 是否懒加载 boolean -
dictFormatter 字典数据格式化，常用于请求字典接口 function(param) -
fetchOptions 全量请求，仅在 data-key="remote" 时有效 function(param) -
remoteSearch 按需请求，输入时执行，仅在 data-key="remote" 且 lazy 时有效 function(query, param) -
remoteEcho 回显请求，初始时执行，仅在 data-key="remote" 且 lazy 时有效 function(values, param) -
param 请求参数 string/object -
remotes 管理全局的请求信息 object -

# props

参数 说明 类型 默认值
value 指定下拉选项的值 string 'value'
label 指定下拉选项的标题 string, function(option): string 'label'
disabled 指定下拉选项是否禁用 string, function(option): boolean -

事件名 说明 回调参数
options-update 选项更新 currentOptions
options-selected 选项选中 selectedOptions

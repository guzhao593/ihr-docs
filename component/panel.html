# API

移除API 1.8.0+ 说明
属性调整： tab-name -> value 由于引入了 collapse 类型，因此再由 tab-name 指定子元素存在歧义
事件调整： tab-update -> input 同理

# Panel 属性

参数 说明 类型 默认值
width 面板宽度 string -
height 面板高度 string -
title 面板标题 string -
toolbar 头部工具栏 array -
shadow 设置阴影显示时机 string: always/hover/never never
border 是否显示外边框 boolean true
type 类型 string: inner/tabs/collapse1.8.0+ -
value/v-model 当类型是 tabs 或 collapse 时用于指定对应的页面 string -
beforeTabLeave 切换标签之前的钩子，若返回 false 或者返回 Promise 且被 reject，则阻止切换。 function(activeName, oldActiveName) -
collapseAccordion1.8.0+ type: collapse 时，是否每次只能展开一个面板 boolean false
empty 是否处于空状态 boolean -
emptyText 设置空状态文本 string -
emptyImgSrc 设置空状态图片路径 string -
contentStyle 设置 content 样式 object -
headerStyle 设置 header 样式 object -
footerStyle 设置 footer 样式 object -

# Panel 事件

事件名 说明 回调参数
tab-click type: tabs 时，tab 被选中时触发 被选中的标签 tab 实例
change 1.8.0+ type: collapse 时，当前激活面板改变时触发(如果是手风琴模式，参数 activeNames 类型为string，否则为array) (activeNames: array / string)

# Panel 插槽

name 说明
- 自定义面板内容
header 自定义面板头部内容
footer 自定义面板底部内容
empty 空状态内容

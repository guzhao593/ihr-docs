# 审计字段组件

该组件主要用于自动填充业务数据库表审计信息，不涉及对SQL的直接修改。

# 功能特性

  • 提供对审计字段created_by，creation_date、updated_by、update_date填充默认实现，对业务代码无侵入；
  • 支持审计字段列自定义，以满足各个项目组个性化需求；
  • 支持审计字段自动填充策略控制，根据SQL类型决定填充相应的字段熟悉；
  • 支持自定义Insert/Update SQL自动填充审计字段；

# 执行过程

  • AuditSourceHolderFilter 拦截用户请求，调用审计数据源解析器生成数据源，并保存到线程变量上；
  • AuditSourceParser 审计数据源解析器，负责解析profile并创建审计数据源，默认实现为DefaultAuditSourceParser；
  • AuditSource 审计数据源，默认实现为DefaultAuditSource;
  • AuditSourceInterceptor myabtis拦截器，拦截Executor#update()，填充审计数据到SQL；
  • AuditSourceHolder 审计数据源线程本地变量

# 快速使用

添加maven依赖到Spring Boot工程即可，无需其他配置，无需修改代码

   <dependency>
      <groupId>com.meicloud.sahara</groupId>
      <artifactId>component-fill-core</artifactId>
      <version>1.0.0-SNAPSHOT</version>
    </dependency>

    try{
        IhrAuditSource<Long> auditSource = new IhrAuditSource<>();
        auditSource.setCreateBy(1L);
        auditSource.setLastUpdateBy(1L);
        AuditSourceHolder.setAuditSource(auditSource);

        // ihr可使用下面的
        // IhrAuditSource.setIhrAuditSource(1L);
    } finally {
        AuditSourceHolder.removeAuditSource();
    }

# 自定义示例

以iHR的审计字段为例

  • 创建自定义审计数据源

    @Data
public class IhrAuditSource<T> implements AuditSource {

    @FillStrategy(FillType.INSERT)
    private T createBy;

    @FillStrategy(FillType.INSERT)
    private Date createDate = new Date();

    @FillStrategy(FillType.ALL)
    private T lastUpdateBy;

    @FillStrategy(FillType.ALL)
    private Date lastUpdateDate= new Date();

    public static <T> void setIhrAuditSource(T id) {
        IhrAuditSource<T> auditSource = new IhrAuditSource<>();
        auditSource.setCreateBy(id);
        auditSource.setLastUpdateBy(id);
        AuditSourceHolder.setAuditSource(auditSource);
    }
}

  • 创建自定义审计数据源对应解析器

    public class IhrAuditSourceParser<T> implements AuditSourceParser {

  @Override
  public AuditSource parse(CommonProfile profile) {

      IhrAuditSource<T> auditSource = new IhrAuditSource<>();
      Object ipeopleProfile = profile.getAttributes().get("ipeopleProfile");
      if (Objects.isNull(ipeopleProfile)) {
          return auditSource;
      }
      T employeeId = (T)(((IpeopleProfile<T>)ipeopleProfile).getEmployeeId());

      auditSource.setCreateBy(employeeId);
      auditSource.setLastUpdateBy(employeeId);
      return auditSource;
  }
}

# 常见问题

  • 为什么自定义SQL没有成功设值？ 审计字段填充组件的职责只是帮忙填充数据，不涉及修改SQL，因此需要自行使用<if>标签添加审计字段到SQL中，然而绝大部分情况下我们是不需要自定义更新或插入SQL的。

      <update id="updateXX">
        update  table
        <set>
            <if test="updated_by != null">
                updated_by = #{updatedBy},
            </if>
            <if test="update_date != null">
                update_date = #{updatedDate},
            </if>
        </set>
        <where>
                ...
        </where>
    </update>
